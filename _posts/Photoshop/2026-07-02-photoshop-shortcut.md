# 📌 화면 제어
{: .notice}

<span class="ul-1"> <span class="color-keyword">Space + LMB 드래그</span> : 캔버스 이동 </span>

<span class="ul-1"> <span class="color-keyword">Alt + Wheel 업/다운</span> : 캔버스 확대/축소 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + R</span> : 눈금자 표시/숨기기 </span>




# 📌 도구 선택
{: .notice}

<span class="ul-1"> <span class="color-keyword">V</span> : Move </span>

<span class="ul-1"> <span class="color-keyword">B</span> : Brush </span>

<span class="ul-1"> <span class="color-keyword">E</span> : Eraser </span>

<span class="ul-1"> <span class="color-keyword">T</span> : Type </span>

<span class="ul-1"> <span class="color-keyword">I</span> : Eyedropper </span>

<span class="ul-1"> <span class="color-keyword">[, ]</span> : 브러쉬 크기 축소, 확대 </span>

<span class="ul-1"> <span class="color-keyword">Q</span> : Quick Mask Mode, 브러쉬로 영역을 직접 선택. 빨간색 영역이 제외되는 영역</span>




# 📌 영역 선택
{: .notice}

<span class="ul-1"> <span class="color-keyword">Shift + LMB 드래그</span> : 영역 추가 </span>

<span class="ul-1"> <span class="color-keyword">Alt + LMB 드래그</span> : 영역 제거 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + D</span> : 영역 선택 해제 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + I</span> : 영역 선택 반전 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + A</span> : 캔버스 전체 영역 선택 </span>




# 📌 컬러
{: .notice}

<span class="ul-1"> <span class="color-keyword">Alt + Delete</span> : 선택된 영역을 Foreground Color 로 채움 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Delete</span> : 선택된 영역을 Background Color 로 채움 </span>

<span class="ul-1"> <span class="color-keyword">X</span> : Fore, Background Color 간 색을 스위칭 </span>

<span class="ul-1"> <span class="color-keyword">1 ~ 0</span> : 무브 툴(V) 상태에서 레이어의 Opacity 빠른 조절 </span>




# 📌 Undo, Redo
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + Z</span> : Undo </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + Z</span> : Redo </span>




# 📌 기타
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + /</span> : 레이어 잠금 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + T</span> : Transform </span>
