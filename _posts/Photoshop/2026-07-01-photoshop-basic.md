# 📌 Selection Brush Tool (L)
{: .notice}

<span class="ul-1"> <span class="color-keyword">Selection Brush Tool</span> : 브러쉬 형태로 영역 선택 </span>

<span class="ul-1"> <span class="color-keyword">Lasso Tool</span> : 올가미 형태로 영역 선택 </span>

<span class="ul-1"> <span class="color-keyword">Polygonal Lasso Tool</span> : 선을 이은 올가미 형태로 영역 선택 </span>

<span class="ul-1"> <span class="color-keyword">Magnetic Lasso Tool</span> : 경계선을 따라 정밀하게 올가미 형태로 영역 선택 </span>




# 📌 Object Selection Tool (W)
{: .notice}

<span class="ul-1"> <span class="color-keyword">Object Selection Tool</span> : 클릭하거나 드래그 해 피사체를 자동으로 선택 </span>

<span class="ul-1"> <span class="color-keyword">Quick Selection Tool</span> : 브러쉬처럼 드래그해 사용, 인접한 영역 중 비슷한 영역을 자동으로 확장해 나갈 수 있다. Shift + 드래그로 영역을 더하거나 Alt + 드래그로 영역을 뺄 수 있다. </span>

<span class="ul-1"> <span class="color-keyword">Magic Wand Tool</span> : 클릭한 지점과 비슷한 색상을 가진 픽셀들을 전부 영역으로 선택 </span>
