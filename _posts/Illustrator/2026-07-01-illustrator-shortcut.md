# 📌 도구
{: .notice}

<span class="ul-1"> <span class="color-keyword">V</span> : 오브젝트 선택, 이동 </span>

<span class="ul-1"> <span class="color-keyword">A</span> : 오브젝트의 개별 앵커 포인트를 선택해 수정 </span>

<span class="ul-1"> <span class="color-keyword">P</span> : 앵커 포인트를 사용해 기존 오브젝트에 추가하거나 오브젝트를 직접 생성 </span>

<span class="ul-1"> <span class="color-keyword">Shift + C</span> : 앵커 포인트를 곡선으로 수정 </span>

<span class="ul-1"> <span class="color-keyword">B</span> : 브러쉬로 자유롭게 선을 그림 </span>

<span class="ul-1"> <span class="color-keyword">I</span> : Eyedropper </span>

<span class="ul-1"> <span class="color-keyword">T</span> : Type </span>

<span class="ul-1"> <span class="color-keyword">G</span> : 오브젝트에 그라데이션 적용 </span>

<span class="ul-1"> <span class="color-keyword">S</span> : 오브젝트 스케일 조절 </span>

<span class="ul-1"> <span class="color-keyword">O</span> : Reflect, 오브젝트 선택 → O → Ctrl + U(스마트 가이드) → 기준점에 Alt + LMB. 복제를 원할 경우 copy를 선택 </span>

<span class="ul-1"> <span class="color-keyword">R</span> : Rotate, 오브젝트 선택 → R → Ctrl + U(스마트 가이드) → 중심축에 Alt + LMB. 복제를 원할 경우 copy를 선택, 톱니바퀴 같은 반복 작업을 해야 할 경우 Ctrl + D(이전 작업 반복)</span>




# 📌 오브젝트 선택, 정렬
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + A</span> : 모든 아트보드의 모든 오브젝트 선택 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + G</span> : 선택한 오브젝트들 그룹화 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + G</span> : 그룹화 해제 </span>

<span class="ul-1"> <span class="color-keyword">오브젝트 더블클릭</span> : 아이솔레이트 모드로 진입해 해당 오브젝트 집중 편집, 빈 공간 더블클릭으로 해제 </span>

<span class="ul-1"> <span class="color-keyword">Shift + F7</span> : Align 패널 </span>




# 📌 레이어
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + [, ]</span> : 오브젝트를 한 단계 뒤, 앞으로 이동 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + [, ]</span> : 오브젝트를 맨 뒤, 앞으로 이동 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + 2</span> : 오브젝트, 레이어를 잠금 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Alt + 2</span> : 잠긴 오브젝트, 레이어의 잠금 해제 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + F</span> : Ctrl + C로 복사한 오브젝트, 레이어를 원래 위치의 Front에 복제 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + B</span> : Ctrl + C로 복사한 오브젝트, 레이어를 원래 위치의 Back에 복제 </span>




# 📌 색상
{: .notice}

<span class="ul-1"> <span class="color-keyword">Shift + X</span> : Fill, Stroke 간 컬러 전환 </span>

<span class="ul-1"> <span class="color-keyword">/</span> : 현재 Fill, Stroke 를 초기화 </span>

<span class="ul-1"> <span class="color-keyword">F6</span> : 컬러 패널 </span>

<span class="ul-1"> <span class="color-keyword">Ctrl + F9</span> : 그라데이션 패널 </span>




# 📌 Pathfinder
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + F9</span> : Pathfinder 패널(도형을 합치거나, 자르거나, 나누는 등의 기능) </span>




# 📌 Undo, Redo
{: .notice}

<span class="ul-1"> <span class="color-keyword">Ctrl + Z</span> : Undo</span>

<span class="ul-1"> <span class="color-keyword">Ctrl + Shift + Z</span> : Redo </span>
